# projeto5

Este projeto consiste em um aplicativo web interativo desenvolvido com Streamlit, cujo objetivo é realizar uma análise exploratória de dados de anúncios de venda de veículos. O aplicativo permite visualizar informações importantes do conjunto de dados por meio de gráficos interativos, facilitando a compreensão dos padrões presentes nos dados.

O conjunto de dados utilizado contém informações como quilometragem (odometer), preço (price) e outras características dos veículos anunciados.

Tecnologias Utilizadas

- Python
- Streamlit
- Pandas
- Plotly Express
